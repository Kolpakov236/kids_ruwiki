О сервисе

Ruwiki Explain — сервис для детей 6–14 лет, который объясняет сложные темы простыми словами с аналогиями, глоссарием и викториной.

Как пользоваться

  • Введите тему в поле ввода или выберите пример
  • Используйте микрофон для голосового ввода 🎤
  • Нажмите «Викторина» чтобы проверить знания 🎮
  • Нажмите «Озвучить» чтобы прослушать ответ 🔊
